# Software libre y licencias

## ¿Qué es el software libre?

El software libre permite utilizar, estudiar, modificar y distribuir el código fuente libremente.

## Tipos de licencias

### Copyleft (GPL)

Permite modificar y distribuir el software, pero obliga a mantener la misma licencia en las versiones derivadas.

Ejemplo:
- Linux Kernel
- GNU Bash

### Licencias permisivas

Permiten modificar y distribuir el software con menos restricciones.

Ejemplos:
- MIT
- BSD
- Apache

### Licencias propietarias

No permiten acceder libremente al código fuente y su uso está restringido por el fabricante.

Ejemplos:
- Microsoft Windows
- Adobe Photoshop

## Licencia de Ubuntu

Ubuntu utiliza principalmente licencias GPL y otras licencias de software libre para los distintos componentes que incorpora.
